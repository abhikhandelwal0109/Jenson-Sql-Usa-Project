# Jenson-Sql-Usa-Project

This project focuses on analyzing business data for Jenson USA using SQL. The objective is to extract meaningful insights related to sales performance, customer behavior, staff efficiency, and product trends.

🚀 Project Overview

In this project, I worked on a structured dataset containing information about products, orders, customers, stores, and staff. Using SQL queries, I solved real-world business problems and generated actionable insights.

🔍 Key Analysis Performed
Analyzed total products sold across different stores
Calculated cumulative sales trends over time
Identified top-performing products by category
Determined the highest spending customers
Found highest-priced products in each category
Measured customer order activity per store
Detected staff members with no sales
Extracted top 3 most sold products
Computed median product pricing
Identified products never ordered
Compared staff performance against average sales
Found customers who purchased from all product categories


🛠️ Tools & Technologies
SQL (MySQL)
Data Analysis Concepts
Joins, Aggregations, Window Functions, Subqueries, EXISTS


📈 Key Insights
Helped identify best-selling products and high-value customers
Highlighted underperforming staff and unused inventory
Provided a clear understanding of sales distribution across stores
Enabled better decision-making using data-driven insights
🎯 Conclusion

This project demonstrates my ability to use SQL for solving complex business problems and performing end-to-end data analysis.
